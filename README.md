# 🔐 SecurePass - Password Strength Checker & Breach Monitoring System

SecurePass is a cybersecurity-focused tool that evaluates password strength and checks whether a password has been exposed in known data breaches. It helps users create stronger, safer passwords and improve their overall online security hygiene.

---

## 🚀 Features

- 🔑 **Password Strength Checker**
  - Analyzes password complexity
  - Provides strength rating (Weak / Medium / Strong)
  - Gives suggestions for improvement

- 🌐 **Breach Detection System**
  - Checks if a password has appeared in known data breaches
  - Alerts users about compromised passwords

- ⚡ **Real-Time Feedback**
  - Instant evaluation of password security
  - User-friendly output for quick understanding
